# Security Policy

## Overview

TrustLens is a lightweight, deterministic AI evaluation tool designed for educational and demonstration purposes. It focuses on transparency, explainability, and responsible AI use.

---

## Supported Versions

This project is provided as a demonstration and may not receive continuous updates or patches.

| Version | Supported |
|--------|----------|
| 1.0    | Yes      |

---

## Reporting a Security Issue

If you discover a potential security issue or vulnerability:

- Please open an issue in this repository
- Clearly describe the issue and steps to reproduce it
- Do not include sensitive data in your report

---

## Responsible Use Disclaimer

TrustLens is an assistive evaluation tool and:

- Does not guarantee factual correctness
- Does not replace human judgment or expert validation
- Does not perform live web verification or external data validation

Users are responsible for verifying information against reliable sources before submission or use.

---

## Scope and Limitations

- The app uses deterministic, rule-based analysis
- It does not access external systems, APIs, or databases
- It does not store or transmit user data
- Results are based solely on input provided by the user

---

## Compliance-by-Design Alignment

TrustLens follows core responsible AI principles:

- **Transparency** – scoring logic is visible and explainable  
- **Explainability** – users can understand why results were generated  
- **Safety** – clear limitations and disclaimers are provided  
- **Validation Awareness** – highlights evidence gaps and uncertainty  
- **Auditability** – outputs include traceable signals and rule triggers
